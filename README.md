## **Installation**

1. Clone the repository:
   ```bash
   git clone https://github.com/username/repo-name.git
   ```
2. Navigate to the project directory:
   ```bash
   cd repo-name
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

---

## **Usage**

1. Start the development server:
   ```bash
   npm run dev 
   ```
2. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```
